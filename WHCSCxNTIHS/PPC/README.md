# PPC

#### Ez Math Calculator

`server` 端會傳送 `500` 行算式，最後才會給你 `flag`

[`pwntools`](https://pypi.org/project/pwntools/) 使用

`python Ez_Math_Calculator.py`

#### Nonono

[nonogram](https://en.wikipedia.org/wiki/Nonogram) 遊戲，解完會給 `QR code`，掃完得到 `flag`

使用 `tools` 或自己刻(?)
